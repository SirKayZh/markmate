隐私政策

MarkMate 是一款本地优先的编辑器。你的数据由你做主。

1. 我们的原则

MarkMate 的设计核心理念是本地优先。你的 Markdown 文档、JSON 文件、配置数据等，全部存储在你的计算机本地文件系统中。MarkMate 不会将你的文档内容上传到任何远程服务器。

2. 我们收集什么

2.1 文档内容

我们不收集、不上传、不存储你的任何文档内容。所有文件读写操作仅在你的本地文件系统进行。MarkMate 没有后端服务器来接收文档数据。

2.2 匿名使用统计（计划中，尚未启用）

未来版本中，MarkMate 可能会在获得你明确同意的情况下，匿名上报聚合使用数据，以帮助我们改进产品。这些数据仅包括：

  • 应用启动次数
  • 打开的文件类型分布（如 .md 占比多少、.json 占比多少）
  • 导出功能使用次数（PDF/HTML/Word 各多少）
  • 崩溃场景与错误类型
  • 使用的操作系统与 MarkMate 版本

我们不会收集：

  • 任何文档内容
  • 文件名或文件路径
  • 个人身份信息
  • IP 地址或其他网络标识

此功能将在首次使用时弹出明确提示，你可以选择同意或拒绝。拒绝后不会有任何数据上报。你可以在设置中随时开启或关闭。

2.3 网站统计

本站点使用 Google Analytics（GA4） 收集匿名访问统计数据，包括页面浏览量、访问来源（搜索引擎/社交媒体/直接访问）、访问者地理位置（仅城市级别）、设备类型与浏览器版本。这些数据帮助我们了解哪些功能更受关注，以便优化网站内容和下载体验。

Google Analytics 通过 Cookie 和设备指纹技术收集以下数据：

  • 页面浏览记录（URL）
  • 访问来源（referrer）
  • 大致地理位置（城市级别，非精确位置）
  • 设备类型、操作系统、浏览器版本
  • 页面停留时间

我们不会通过 Google Analytics 收集：

  • 个人身份信息（姓名、邮箱、IP 地址等）
  • 精确地理位置（GA4 默认匿名化 IP）

你可以通过浏览器设置禁用 Cookie 或安装广告拦截插件来阻止 Google Analytics 追踪。Google 的数据处理条款详见其 隐私政策。

此外，本站也使用浏览器本地存储（localStorage）记录页面访问与下载点击行为。这些数据仅存储在你的浏览器中，不会被发送到任何服务器。

3. 第三方服务

MarkMate 应用本身不集成任何第三方分析、广告或追踪 SDK。MarkMate 网站托管在 GitHub Pages 上，GitHub 可能会按照其隐私声明收集访问日志。

本站使用 Google Analytics（GA4） 进行网站访问统计。Google Analytics 是 Google LLC 提供的网络分析服务，通过 Cookie 和 Web 信标收集匿名访问数据。Google 可能会将这些数据用于其自身的服务改进。详情请参阅 Google 隐私政策 和 Google Analytics 数据保护说明。

4. 数据安全

由于你的文档完全存储在本地，数据安全取决于你的计算机安全实践。我们建议：

  • 定期备份重要文档
  • 使用操作系统提供的全盘加密功能（如 macOS FileVault、Windows BitLocker）
  • 不要在不信任的计算机上打开敏感文档

5. 开源透明

MarkMate 是完全开源的（MIT 协议）。所有源代码公开在 GitHub。你可以自行审查代码，确认没有隐藏的数据收集行为。欢迎提交 Issue 或 PR 对隐私实践提出改进建议。

6. 联系我们

如果你对隐私政策有任何疑问或建议，请通过以下方式联系：

  • 提交 GitHub Issue
  • 填写反馈问卷

7. 政策更新

本隐私政策可能会随产品迭代而更新。重大变更将通过 GitHub Release Notes 和本页面公告。
