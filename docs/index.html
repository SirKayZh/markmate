✨ 功能特性

✍️

所见即所得 Markdown

Vditor IR 即时渲染，打字即预览。专注模式 + 打字机模式 + 5 套排版主题，沉浸写作。

💬

JSONL 对话视图

自动识别 OpenAI / Alpaca / ShareGPT 格式，角色着色气泡（system·user·assistant·tool）。统计面板一眼看清轮次与 Token。

🏷️

批量标注编辑

点击气泡直接改，攒多了一起保存。翻页不丢编辑，失败条目红框高亮定位。

🧭

多 Tab + 文件管理

⌘T 新建 / ⌘W 关闭 / ⌘1-9 切换。侧栏收藏夹 + 最近打开 + 当前文件夹，⌘P 快速打开。

💻

JSON / YAML / XML

Prism.js 语法高亮，⌥⌘L 一键格式化。可折叠 JSON 树视图 + 大文件异步 Worker 解析。JSON 报错精准定位到行列。

📤

多格式导出

一键导出 PDF / HTML / Word（.docx）/ 长图 PNG。保留字体、代码块、表格、图片。

🛡️

永不丢稿

1.5s 自动保存 + 每文件 10 个历史快照 + 草稿恢复 + 关闭确认。三重保险。

🔄

自动更新

启动后静默检查 GitHub Release，发现新版弹出横幅，一键下载 + 重启安装（v2.0.0+）。

🎨

主题与布局

亮色/暗色/跟随系统 + 5 套排版主题。三栏可拖动布局（大纲↔编辑器↔源码），双向滚动同步。
