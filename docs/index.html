✨ 功能特性

✍️

所见即所得编辑

Vditor IR 即时渲染，打字即预览，体验如 Typora。专注模式 + 打字机模式，沉浸写作。

🧭

智能导航

多级大纲树 + 文件管理面板（收藏/最近/当前文件夹）。⌘P 快速打开，模糊搜索秒开文件。

🛡️

永不丢稿

1.5s 防抖自动保存 + 每文件 10 个历史快照 + 崩溃恢复。未保存关闭时弹窗确认，三重保险。

📤

多格式导出

一键导出 PDF / HTML / Word（.docx）/ 长图 PNG。保留字体、代码块、表格、图片，零外部依赖。

💻

代码/配置文件查看器

直接打开 .json / .xml / .yaml / .jsonl，Prism.js 语法高亮，一键格式化。轻量替代 VS Code 查看。

🎨

主题与布局

亮色/暗色/跟随系统 + 5 套排版主题。三栏可拖动布局（大纲↔编辑器↔源码），双向同步。
